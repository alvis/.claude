What the simulation teaches

Cohort size changes blast radius, not correctness. When backfill is incomplete, some new reads land on rows that only the old shape can answer — the fallback turns those from failures into quiet legacy reads. The authority switch is safe only when that amber band reaches zero and the rollback path has been exercised.
