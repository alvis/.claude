Author's own cuts

What this ranking deliberately left out.

These are the reviewer's non-goals and soft spots — not the costs of any one direction, which live in each frame's pressure case. Read them before treating the order as settled.

Left out on purpose

  • No big-bang cutover was scored. The independent-deploy constraint rules it out before ranking, so it never appears as a fifth frame.
  • Ranked on rollback, then ownership, then elegance only. Latency and dollar cost were not scored because no option is disqualified on either at six consumers and this volume.
  • One representative pressure case per option, not an exhaustive failure catalogue. Each frame shows the failure most likely to change the rank, not every way it can break.

Weakest part of this board

  • The rank 3 versus rank 4 order is the softest call. If any single consumer truly needs an independent rollback, consumer translators should rise — and this board does not resolve that.
  • The replay-equality and lag figures are illustrative shapes for inspecting each mechanism, not measured throughput. They are flagged as such in the generated brief and must not size infrastructure.
