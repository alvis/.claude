# Coding specialist routing

| Tasks | Route to |
| --- | --- |
| Bootstrap and scaffold a new project | `project-initializer` |
| Author tests via TDD | `testing-evangelist` |
| Implement a general library, data pipeline, CLI, or glue code | `generalist-engineer` |
| Automate CI/CD and infrastructure | `devops` |
| Validate an exploit with an adversarial PoC | `adversarial-red-team` |
| Review changed code for quality, including day-to-day security review | `code-quality-critic` |
| Debug hard bugs, optimize performance, or crack algorithms | `principal-engineer` |
| Deep security review, only when explicitly requested (most experienced, most expensive) | `security-champion` |
| Run a lint, type, and test sweep and summarize it | `test-runner` |
