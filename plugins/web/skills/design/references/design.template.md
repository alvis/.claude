This entrypoint indexes the complete durable design template. Load and complete
every applicable child in order:

1. [Theme, palette, and typography](design-template/10-theme-color-and-typography.md)
2. [Components, layout, and depth](design-template/20-components-layout-and-depth.md)
3. [Rules, responsive behavior, and agent prompts](design-template/30-rules-responsive-and-prompts.md)
4. [Context, inventory, implementation state, and file map](design-template/40-handoff-contract.md)

The children form one design contract. Sections 10–13 in the final child are
the required domain handoff surface; keep them synchronized with implementation
state rather than copying their detail into this overview.
