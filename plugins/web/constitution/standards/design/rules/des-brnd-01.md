# DES-BRND-01: Brand Personality

## Intent

Brand personality must be visible and consistent through color, typography, and tone. Design must feel current and modern — not generic templates or dated patterns. Restrained color, spacious layout, and typography-led hierarchy signal modern design.

## Fix

- Express brand through: primary color, typography choice, tone of voice in microcopy
- Use restrained color palette — let whitespace and typography lead
- Spacious layout with generous padding and breathing room
- Avoid dated patterns: heavy gradients, skeuomorphism, bevels, glossy effects
- Consistent brand expression across all pages and components
- Modern aesthetic: clean lines, subtle shadows, thoughtful color application

## Code Superpowers

- Check for brand color token usage (--color-primary, --color-accent)
- Verify consistent font-family usage across pages
- Flag dated CSS patterns (heavy gradients, text-shadow for 3D effect, excessive border styling)

## Common Mistakes

1. Generic/template appearance with no brand personality (default Bootstrap look)
2. Dated visual patterns (heavy gradients, skeuomorphism, glossy buttons)
3. Brand colors used inconsistently across screens
4. Cluttered layout with no breathing room

## Edge Cases

- Some industries (banking, healthcare) may intentionally use conservative design — but it should still feel intentional, not template-derived
- Retro/nostalgic design can be intentional brand expression when done consistently
