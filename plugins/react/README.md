# React

React component development standards routed over the shared Coding
workflows. Depends on `coding` and `essential`. React owns the framework
rules (JSX, components, hooks, accessibility, project structure, tests,
Storybook conventions) under `constitution/standards/`; Coding owns the
generic execution and reporting they run through.

## Skills

| Skill | Use when |
| --- | --- |
| `react:react` | Creating, editing, reviewing, or routing any React/JSX/hooks/component work — selects the React standards for the task. |
| `react:lint` | Mechanical enforcement of the React standards through the shared `coding:lint` workflow. |
